1. Create an empty list to keep track of each page and how many times it is visited.
2. If the page is already in the list, increase its visit number by one.
3. If the page is not in the list, add it and set its visit number to one.
4. Arrange all the pages so that the ones with the highest visit numbers come first.
5. Take the first five pages from this arrangement.
6. Show these five pages and how many times each has been visited.
